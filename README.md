# 📊 Global Developer Technology Trends Analysis

### Data Analyst Capstone Project

This repository contains my **Data Analyst Capstone Project**, completed as part of the **IBM Data Analyst Professional Certificate**.

The project analyzes global developer survey data to identify **technology usage patterns, emerging technology trends, and developer demographics**. The analysis follows the complete **data analytics workflow**, from data preparation and exploratory analysis to dashboard visualization and insight communication.

The goal of the project is to demonstrate how **data-driven analysis can reveal trends in programming languages, databases, platforms, and developer behavior across the global technology ecosystem.**

---

# 📁 Project Overview

This project explores key questions such as:

* Which programming languages are most commonly used by developers today?
* Which technologies are developers most interested in learning next?
* What database technologies dominate current and future demand?
* What are the demographic characteristics of developers participating in the survey?

By analyzing these patterns, the project provides insights that can help **developers, employers, and technology leaders better understand evolving industry trends.**

# 🔄 Project Workflow

## 1. Data Collection

The analysis uses developer survey datasets provided within the IBM Data Analyst course environment.

## 2. Data Cleaning and Preparation

Data preparation was performed using **Python and Pandas**, including:

* Handling missing values
* Removing duplicate records
* Correcting data types
* Preparing fields for analysis

These steps ensured the dataset was suitable for accurate analysis.

## 3. Exploratory Data Analysis (EDA)

Exploratory analysis was conducted to understand trends in:

* Programming languages
* Database technologies
* Web frameworks
* Development platforms
* Developer demographics

Statistical summaries and visualizations were produced using **Matplotlib and Seaborn**.

## 4. Dashboard Development

Interactive dashboards were created using **IBM Cognos Analytics** to visualize the most important insights.

The dashboard consists of three main sections:

### **Current Technology Usage**

* Top programming languages developers currently use
* Most widely used databases
* Popular development platforms
* Common web frameworks

### **Future Technology Trends**

* Programming languages developers want to work with
* Databases expected to grow in demand
* Emerging technology platforms

### **Developer Demographics**

* Age distribution of developers
* Geographic distribution
* Education levels

These dashboards help transform raw survey data into **clear and interactive visual insights**.

## 5. Reporting and Presentation

The final stage of the project involved presenting the analysis in a **structured capstone presentation**, summarizing key trends and implications for the technology industry.

---

# 📈 Key Insights

Several important insights emerged from the analysis:

* **Full-stack web technologies** remain the most widely used development stack.
* **PostgreSQL** appears as one of the most dominant and rapidly growing database technologies.
* **Cloud platforms such as AWS and Azure** continue to gain strong developer adoption.
* The developer community represented in the dataset is **globally distributed and largely concentrated within the 25–34 age group**.

These findings illustrate how data analytics can help identify **technology adoption patterns and future industry directions.**
